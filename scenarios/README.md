# Scenario Planning Cards

Pre-implementation planning documents for all 100 scenarios.

Each card defines the **problem, math model, and key parameters** before coding begins.
Once a scenario is implemented, its full results (code output, visualizations) live in [`outputs/`](../outputs/).

## vs. `domains/`

| Folder | Purpose |
|--------|---------|
| `domains/` | Theory background and overview for each of the 5 research domains |
| `scenarios/` | Per-scenario specs: problem definition, math, parameters |
| `outputs/` | Completed scenarios: results, plots, animations |
